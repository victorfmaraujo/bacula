Script que facilita aqueles que gostam de "quebrar" os parâmetros do Bacula em diversos arquivos

baixe o script, dê permissão de execução e rode
Obs:  o script já está com o caminho padrão do bacula que é /etc/bacula, caso a sua instalação seja diferente, modifique isso antes de rodar o script.

wget https://raw.githubusercontent.com/victorfmaraujo/bacula/master/modbacula.sh
chmod +x ./modbacula.sh
./modbacula.sh
